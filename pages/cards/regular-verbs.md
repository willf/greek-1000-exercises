## Rules about vowel contractions

### -εω

1. ε·ε → ει
2. ε·ει → ει
3. ε·ο → ου
4. ε·οι → ου
5. ε·ω → ω

### -αω

1. α·ε → α
2. α·ει → ᾳ
3. α·ο → ω
4. α·ου → ω
5. α·ω → ω

### Intervocalic sigma

1. VσV → VV (and then other vowel contractions)

## Rules about vowel augments in the aorist

1. ε·α → η
2. ε·ε → η
3. ε·η → η
4. ε·ι → ι
5. ε·υ → υ
6. ε·ο → ω
7. ε·ω → ω
8. ε·ει → ῃ
9. ε·οι → ῳ
